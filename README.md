# Habit Mastery League

A modern Flutter-based habit tracking application designed to help users build consistency, track progress, and improve daily productivity through gamification and analytics.

## Features
- Add and manage daily habits
- Track habit completion
- XP, Level, and Rank system
- Weekly analytics dashboard
- Habit category breakdown
- Light and Dark theme support
- Local data persistence using SQLite
- Notification reminders
- Clean and modern UI

## Tech Stack
- Flutter
- Riverpod
- SQLite (sqflite)
- GoRouter
- SharedPreferences
- FL Chart
- Flutter Local Notifications

## Screens
- Home Screen
- Analytics Screen
- Settings Screen
- Habit Detail Screen
- Edit Habit Screen

## Architecture
The app follows a feature-based clean architecture:
- Presentation layer
- Domain layer
- Data layer
